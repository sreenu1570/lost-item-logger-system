📦 About the Project
The Lost Item Logger System for College Campus (LIL-CAMP) is a Python-based application developed to efficiently manage lost and found items within a college environment. The system provides a simple and user-friendly interface where users can report lost items, log found items, search for items, and claim them.

The application uses a CSV file as a lightweight database to store and retrieve data, ensuring persistence and easy access. The project is designed using modular programming, where different functionalities such as adding items, searching, viewing, and claiming are handled in separate modules.

This system helps reduce confusion and manual effort in tracking lost belongings on campus and provides a structured approach to item management.

🎯 Objectives
To develop a simple system for managing lost and found items
To store and retrieve data using CSV file handling
To implement modular programming concepts in Python
To provide an efficient and user-friendly solution for campus use
🛠️ Key Features
Add Lost Items
Add Found Items
Search Items by Name
View Lost and Found Records
Claim Found Items
Persistent Data Storage using CSV
